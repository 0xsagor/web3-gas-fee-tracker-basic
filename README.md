# Web3 Gas Fee Tracker Basic

This project helps beginners learn how gas fees work on blockchain networks.

You will learn:
- What gas fees are
- How to connect to a blockchain provider
- How to read current gas price
- How to display network data
- How Web3 dashboards work

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-gas-fee-tracker-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage  
- app.js = Frontend logic  
- gas.js = Gas fee functions  
- web3.js = Blockchain connection  
- config.js = Network settings  
- server.js = Local server  
- package.json = Project info  
- utils.js = Helper functions  
- .env.example = Environment example  

## Features
- Show current gas price  
- Refresh gas data  
- Simple dashboard  
- Network info display  

## Learning Goal
This project explains how gas fees change on blockchain networks.
